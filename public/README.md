# 📊 竞品情报简报 AI Agent

> **Autonomous AI Agent** · 每周自动搜集竞品动态 → AI 分析 → 飞书推送  
> 一个从零搭建的生产级 Agent 项目

---

## 🤖 这是什么

一个**真正自主工作的 AI Agent**，不是脚本，不是爬虫。

每周一早上 09:00，Agent 会自动：
1. 读取配置的竞品列表
2. 搜索全网最新动态（GitHub Release、技术新闻、社区讨论）
3. 用 LLM 分类、提炼、判断影响
4. 生成结构化竞争情报简报
5. 推送到飞书私聊

**全程零人工干预。** 加一个新竞品？改一行 YAML 配置就行。

---

## 🏗 架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  配置层      │    │  执行层     │    │  交付层     │
│             │    │             │    │             │
│ config.yaml │───▶│ Cron Agent  │───▶│ Heartbeat   │───▶ 飞书
│ (竞品/关键词)│    │ (搜索/分析)  │    │ (文件信号)  │
└─────────────┘    └─────────────┘    └─────────────┘
```

**关键设计：**
- **执行与交付解耦** — Agent 在隔离环境执行，通过文件信号触发主会话推送
- **中英双语搜索** — 每个竞品覆盖国内外动态
- **结果去重** — 跨关键词自动合并
- **部分容错** — 单个竞品失败不影响整体
